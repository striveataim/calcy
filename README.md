# Calcy - CGPA & Grade Calculator

A modern Next.js web application for calculating CGPA, SGPA, and grades for Indian universities.

## Features

- 📊 **CGPA ↔ Percentage Conversion**
- 🎓 **9 University-Specific Calculators** (VTU, SPPU, Anna, Mumbai, Delhi, Chandigarh, Manipal, HEC Pakistan, CBSE)
- 📐 **SGPA → CGPA Converter**
- 📝 **Semester Calculator**
- 🏆 **Division/Class Finder**
- 📱 **Mobile Responsive**
- ⚡ **Fast & Offline**

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack

- Next.js 15+ with App Router
- Tailwind CSS
- TypeScript
- Node.js 18+
